  Tarea diaria<br />

       ```sql
       select numportabilidad "Numero movil", numcuenta "Código Cuenta", 'Inmediato' as "Modo Negocio" from vpap.vpap.tb_rpa_convergent_sales where cuentaportabilidad = false order by id;
       ```
